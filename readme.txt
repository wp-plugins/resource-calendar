== Description ==

Resource Calendar enables the reservation of resources.  
Resource Calendar will be useful for a need for displaying the availability of a resource.  

== Installation ==

1. Upload `resource-calendar` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
